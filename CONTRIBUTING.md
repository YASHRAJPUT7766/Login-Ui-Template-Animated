# Contributing to Style Lovin Logins 🎨

## How to Add a Design

1. Fork this repo
2. Create folder in `login-designs/your-design-name/`
3. Add `index.html`, `style.css`, (optional) `script.js`
4. Submit a Pull Request

## Naming Convention

- ✅ `glassmorphism-dark`
- ✅ `retro-terminal-login`
- ❌ `MyLogin`, `login 1`

## Design Rules

- Self-contained in its folder
- No external API keys
- Works by opening index.html directly
- Include a `preview.png` screenshot if possible

## Code Style

- Indent with 2 spaces
- Comment your clever CSS tricks
- Mobile responsive preferred

## 🚀 What is this?

`style-lovin-logins` is a curated collection of **1000+ login page designs** — each one handcrafted with a unique aesthetic. Glassmorphism to brutalist, retro terminal to luxury editorial.

## 📁 Folder Structure

```
style-lovin-logins/
├── login-designs/         ← All 1000+ designs go here
├── assets/                ← Shared fonts, icons, images
├── docs/                  ← Documentation & guides
├── scripts/               ← Dev & automation scripts
├── tests/                 ← UI/accessibility tests
├── .github/               ← GitHub Actions, issue templates
└── config/                ← Project config files
```

## ⚡ Quick Start

```bash
git clone https://github.com/style-lovin/style-lovin-logins.git
cd style-lovin-logins
npm install
npm run dev
```

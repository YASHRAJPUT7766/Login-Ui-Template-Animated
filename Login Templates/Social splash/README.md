## 🌌 Design Philosophy

<div align="center">

```
╔══════════════════════════════════════════════════════════╗
║                                                          ║
║   💙  Clean, human-first interface — always              ║
║   🚫  No AI clutter — only meaningful UI                 ║
║   ✨  Micro-interactions that feel alive                 ║
║   🌗  Dual theme — dark & light, zero flicker            ║
║   🎯  Every pixel is intentional                         ║
║                                                          ║
╚══════════════════════════════════════════════════════════╝
```

</div>

> Social login UIs are often boring, bloated, or just... ugly.  
> **social-splash** fixes that — with animations that feel tactile,  
> a theme toggle that's instant, and a UI so clean it could ship tomorrow.

## 🎨 Customization

Want to make it yours? It's built for easy theming:

```css
:root {
  --facebook:   #1877f2;   /* Facebook brand blue */
  --check-fill: #7c6ff7;   /* Checkbox accent */
  --accent2:    #bba9ff;   /* Secondary highlights */
  --bg-dark:    #0f0c29;   /* Dark mode background */
  --bg-light:   #f4f4ff;   /* Light mode background */
}
```

**➕ Add a new provider button:**
```html
<button class="btn-provider" data-provider="GitHub">
  <svg><!-- GitHub icon --></svg>
  Continue with GitHub
</button>
```

**✏️ Edit the wordmark / logo text:**
```html
<span class="wordmark">YourBrand</span>
```

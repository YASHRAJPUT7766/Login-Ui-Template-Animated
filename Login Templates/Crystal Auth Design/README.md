## 💎 Design Philosophy

<div align="center">

```
╔══════════════════════════════════════════════════════════╗
║                                                          ║
║   ✦   Dark luxury aesthetic — obsidian & gold            ║
║   💎  Live 3D crystal gem — Three.js powered             ║
║   🖱️   Custom magnetic gold cursor                        ║
║   ⚡  Every animation is intentional & smooth            ║
║   🎯  Two-panel layout — visual story + form             ║
║   🔒  Real validation — not just pretty UI               ║
║                                                          ║
╚══════════════════════════════════════════════════════════╝
```

</div>

> Auth forms are forgotten UX. **crystal-auth** makes sign-in a moment —  
> a spinning gem, gold micro-interactions, and a form so refined  
> it feels like logging into something that matters.

## 📁 Project Structure

```bash
crystal-auth/
│
├── 📄 index.html     ← Semantic HTML only — clean & readable
├── 🎨 style.css      ← All variables, layout, animations
├── ⚡ script.js      ← Three.js gem · cursor · validation · forms

```

> ✅ **Fully separated** — HTML, CSS, JS in their own files.  
> No inline styles. No inline scripts. Clean architecture.

## 🎨 Customization

Dead-easy theming via CSS variables:

```css
:root {
  --gold:        #c9a84c;   /* Primary gold accent */
  --gold-light:  #e8ca7a;   /* Hover highlight */
  --gold-dim:    #8a6e30;   /* Muted gold */
  --dark:        #0c0b09;   /* Deepest background */
  --dark-3:      #1e1c18;   /* Form panel bg */
  --dark-4:      #252220;   /* Input bg */
  --cream:       #fafaf8;   /* Primary text */
}
```

**🔮 Change gem color:**
```javascript
// In script.js — find gemMat and tweak emissive:
emissive: 0x4c88c9,          // Blue crystal
emissiveIntensity: 0.06,
```

**➕ Add a provider:**
```html
<button class="btn-social">
  <svg><!-- icon --></svg>
  Discord
</button>
```

## ⚡ Design Philosophy

<div align="center">

```
╔══════════════════════════════════════════════════════════╗
║                                                          ║
║   ⚡  Clean, minimal aesthetic — white card on mesh      ║
║   🏃  Submit button physically escapes bad input         ║
║   🪪  Floating logo ring — drop your brand above card    ║
║   🗂️  3-tab system — Sign In · Sign Up · Reset           ║
║   🔒  Real validation — not just pretty UI               ║
║   📢  Toast feedback — social auth & success states      ║
║                                                          ║
╚══════════════════════════════════════════════════════════╝
```

</div>

> Auth forms are boring. **chaos-auth** makes sign-in memorable —  
> a button that runs from bad input, live strength feedback,  
> and a form clean enough to ship today.

## 🎨 Customization

Dead-easy theming via CSS variables:

```css
:root {
  --bg:      #f0f2f8;   /* Page background */
  --card:    #ffffff;   /* Card background */
  --accent:  #5b4fff;   /* Primary brand colour */
  --err:     #e53e5a;   /* Error red */
  --ok:      #16a97a;   /* Success green */
}
```

**🪪 Swap the logo icon:**
```html
<!-- In index.html — find .logo-inner and replace the SVG: -->
<div class="logo-inner">
  <img src="your-logo.png" width="28" height="28" alt="Logo"/>
</div>
```

**➕ Add a social provider:**
```html
<button class="btn-social" data-provider="Discord">
  <svg><!-- icon --></svg>
  Discord
</button>
```

# Creating a Login Design

## Step 1: Create Your Folder

```bash
mkdir login-designs/my-awesome-login
cd login-designs/my-awesome-login
```

## Step 2: Create index.html

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Awesome Login</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <div class="login-container">
    <h1>Welcome Back</h1>
    <form>
      <input type="email" placeholder="Email">
      <input type="password" placeholder="Password">
      <button type="submit">Sign In</button>
    </form>
  </div>
  <script src="script.js"></script>
</body>
</html>
```

## Step 3: Style It

Go wild in `style.css`. Import Google Fonts, use CSS variables, animations — anything!

## Step 4: Take a Screenshot

Save as `preview.png` (1280x720 recommended).

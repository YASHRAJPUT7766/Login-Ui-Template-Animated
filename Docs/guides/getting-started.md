# Getting Started

## Prerequisites

- Git
- Node.js v20+
- A browser (Chrome recommended for testing)

## Setup

```bash
# 1. Clone the repo
git clone https://github.com/style-lovin/style-lovin-logins.git
cd style-lovin-logins

# 2. Install dev tools
npm install

# 3. Preview any design
open login-designs/glassmorphism-dark/index.html

# 4. Or run the preview server
npm run dev
```

## Folder Structure

Each design lives in its own self-contained folder:

```
login-designs/
└── my-design-name/
    ├── index.html     ← Required
    ├── style.css      ← Required
    ├── script.js      ← Optional
    └── preview.png    ← Recommended
```

# Naming Convention

## Folder Names

Use `kebab-case` — all lowercase, words separated by hyphens.

### Format

`[style]-[variant]` or `[adjective]-[noun]-[optional-detail]`

### Examples

| Good ✅ | Bad ❌ |
|---|---|
| `glassmorphism-dark` | `GlassMorphism` |
| `retro-terminal-green` | `login 1` |
| `minimal-split-screen` | `FINAL_v2` |
| `brutalist-bold-red` | `myLogin` |
| `luxury-editorial-cream` | `login_design` |

## Category Prefixes

| Prefix | Style |
|---|---|
| `glass-` | Glassmorphism |
| `neo-` | Neumorphism |
| `retro-` | Retro / Terminal |
| `minimal-` | Minimal / Clean |
| `dark-` | Dark themed |
| `light-` | Light themed |
| `gradient-` | Gradient heavy |
| `3d-` | 3D effects |
| `brutal-` | Brutalist |
| `luxury-` | Luxury / Editorial |
